## 第1章：Big Five と音声特徴量の科学的相関

### 1-1. 各因子の音声特徴量との対応表

| Big Five 因子 | 主要な音声特徴量 | 相関の強さ | 方向性 |
|---|---|---|---|
| **開放性 (Openness)** | F0の標準偏差・声の多様性・イントネーション豊かさ | 小〜中 | 高スコア → F0変動大、豊かなイントネーション |
| **誠実性 (Conscientiousness)** | 話速・シマー・CPP | 小〜中 | 高スコア → 速い話速、明瞭な発話 |
| **外向性 (Extraversion)** | **音量（RMS）・F0変動・話速・ポーズ少** | **中〜強（r≈0.38〜0.40）** | 高スコア → 大きな声、速い話速、活発なピッチ変動 |
| **協調性 (Agreeableness)** | 抑揚変化量・音調のダイナミクス・スペクトル重心 | 小〜中 | 高スコア → 柔らかい音調、豊かな抑揚 |
| **神経症傾向 (Neuroticism)** | **F0不安定性・ジッター・シマー・ポーズ多・話速遅** | **中〜強（r≈0.38〜0.40）** | 高スコア → 声帯振動不安定、言い淀み増加 |

### 1-2. 特徴量ごとの詳細解説

#### 🔵 基本周波数（F0 / ピッチ）

```
男性 正常域：80〜150 Hz
女性 正常域：200〜220 Hz
```

| F0 の特性 | 関連する性格・印象 | 科学的根拠 |
|---|---|---|
| **F0が低い（男性）** | 支配性・権威・リーダーシップ・信頼性 | 政治討論研究等で確認。非言語的優位性シグナル |
| **F0が高い** | 外向性（一部）・感情的反応性 | 外向性との相関あり（効果量は小〜中） |
| **F0変動が大きい** | 外向性・開放性・感情豊か | ピッチレンジと外向性の正の相関（r≈0.15〜0.30） |
| **F0変動が小さい（単調）** | 内向性・低開放性 | 単調な声 → 内向的・感情抑制的 |
| **F0が不安定** | 神経症傾向・不安・ストレス | ジッター上昇と連動 |

#### 🔵 話速・ポーズ（Tempo & Pauses）

| 特性 | 関連する性格 |
|---|---|
| **話速が速い・流暢** | 外向性・誠実性（自信ある発話として知覚） |
| **話速が遅い** | 神経症傾向・内向性 |
| **ポーズが多い・長い** | 神経症傾向（ためらい・言い淀み・発話のつまずき） |
| **ポーズが少ない** | 外向性（流暢・スムーズな発話） |

#### 🔵 ジッター（Jitter）— 周波数摂動

```
正常値：1.0%以下（高精度基準：0.5%以下）
```

| ジッター値 | 意味 | 関連状態 |
|---|---|---|
| 低い（<0.5%） | 声帯振動が安定 | 心理的安定・誠実性・低神経症傾向 |
| 中程度（0.5〜1.0%） | やや不安定 | 通常範囲内 |
| 高い（>1.0%） | 声帯振動が不規則 | 不安・ストレス・神経症傾向の高さ |
| 非常に高い（>2〜3%） | 異常レベル | 声帯病理・重篤なストレス状態 |

注意：ジッターは主に声帯健康状態の指標。性格との直接相関は間接的（自律神経→声帯筋緊張→ジッター）

#### 🔵 シマー（Shimmer）— 振幅摂動

```
正常値：3.8%以下（高精度基準：3.0%以下）
```

| シマー値 | 意味 | 関連状態 |
|---|---|---|
| 低い（<3.0%） | 音量が安定 | 心理的安定・自信・低ストレス |
| 高い（>3.8%） | 音量が不安定 | ストレス・疲労・不安・うつ傾向 |

研究知見：機能性嗄声患者研究で不安とシマーの有意な相関、うつと両指標の相関が確認された

#### 🔵 HNR（ハーモニクス対ノイズ比）

```
正常値：15 dB以上（高いほど良い）
```

| HNR値 | 声の質 | 関連状態 |
|---|---|---|
| 高い（>20 dB） | 澄んだ・クリアな声 | 心理的健康・低ストレス・外向性 |
| 中程度（15〜20 dB） | 標準 | 通常状態 |
| 低い（<15 dB） | ノイズが多い・かすれ声 | ストレス・うつ・疲弊状態 |

デジタルバイオマーカーとして研究中：うつ病・不安症の検出にHNRを含む多次元特徴量が活用されている

#### 🔵 音量・RMSエネルギー

| 特性 | 関連する性格 |
|---|---|
| **音量が大きい・高RMS** | **外向性（最も信頼性の高い指標）** |
| **音量の変動が大きい** | 外向性・開放性 |
| **音量が小さい・低RMS** | 内向性 |
| **音量が安定して大きい** | 支配性・自信 |

最新研究（2023年）：ピッチ変動よりも声の大きさのほうが外向性のより信頼できる指標

## 第2章：周波数帯域と性格・感情の対応

### 2-1. 周波数帯域別の性格・感情対応マップ

| 周波数帯域 | 主な音響成分 | 性格・感情との関連 |
|---|---|---|
| **低域 (100〜500 Hz)** | 基本周波数 F0・倍音の基盤成分 | 支配性・権威・自信・外向性。低F0 → リーダーシップ的印象 |
| **中域 (500〜2000 Hz)** | F1・F2 フォルマント・声の「温かさ」 | 社会的知覚（有能さ・支配性）の主要決定因子 |
| **高域 (2000 Hz以上)** | F3・高次倍音・子音 | 声の「明るさ」「鮮明さ」。外向性・誠実性との正の関連 |

### 2-2. スペクトル重心（Spectral Centroid）と性格・感情

| スペクトル重心 | 音の印象 | 感情・心理的関連 |
|---|---|---|
| **高い（明るい声）** | 明るい・輝き・高周波エネルギーが豊富 | 高覚醒（怒り・喜び・興奮）・支配的存在感 |
| **低い（暗い声）** | 暗い・重厚・低周波エネルギー優位 | 低覚醒（悲しみ・退屈・うつ）・服従的傾向 |

## 第3章：実装に使う特徴量セットと測定基準

### 3-1. VOICECODEで使用するコア音声特徴量一覧

| # | 特徴量 | 測定値 | Big5への寄与 |
|---|---|---|---|
| 1 | F0平均 | Hz | 外向性（支配性側面）・神経症傾向 |
| 2 | F0標準偏差 | Hz | 外向性・開放性 |
| 3 | F0最大値 | Hz | 感情的反応性・神経症傾向 |
| 4 | 音量（RMS）平均 | dB | **外向性（最重要）** |
| 5 | 音量変動 | dB | 外向性・開放性 |
| 6 | 話速 | 音節/秒 | 外向性・誠実性 |
| 7 | ポーズ比率 | % | 神経症傾向（反比例） |
| 8 | ジッター（local） | % | 神経症傾向・ストレス |
| 9 | シマー（local） | % | 神経症傾向・ストレス |
| 10 | HNR | dB | 心理的健康・外向性 |
| 11 | スペクトル重心 | Hz | 覚醒度・外向性 |
| 12 | MFCC 1〜13係数 | - | 声の質感・声道特性の包括的表現 |

### 3-2. 推奨ツールスタック

Python ライブラリ:
- librosa : 基本的な音声解析（F0・MFCC・スペクトル重心等）
- parselmouth : PRAATのPythonラッパー（ジッター・シマー・HNR・フォルマント）
- opensmile : GeMAPS 62次元の標準特徴量セット（Big5予測実績多数）

openSMILE 使用例:
```python
import opensmile
smile = opensmile.Smile(
    feature_set=opensmile.FeatureSet.GeMAPSv01b,
    feature_level=opensmile.FeatureLevel.Functionals,
)
features = smile.process_file('audio.wav')
```

### 3-3. 録音品質の最低基準

| 項目 | 推奨値 |
|---|---|
| サンプリングレート | 16 kHz以上（推奨：44.1 kHz） |
| 録音レベル | 平均 -12 dBFS、ピーク -6 dBFS以下 |
| マイク距離 | 15〜30 cm |
| 録音時間 | 30秒以上（推奨：1〜3分の自由発話） |
| 環境 | 静粛室（エアコン音・反響なし） |
| ファイル形式 | WAV推奨（非圧縮） |

## 第4章：診断ロジックの設計フレームワーク

### 4-1. 処理パイプライン

音声ファイル入力
→ 前処理（ノイズ除去・正規化・無音区間除去）
→ 特徴量抽出（GeMAPS 62次元 + カスタム特徴量）
→ 性別補正（F0・フォルマントを性別ごとに正規化）
→ Big5スコア推定（外向性・神経症傾向・開放性・誠実性・協調性）
→ スピリチュアル表現への変換（VOICECODEオリジナル解釈エンジン）
→ PDF診断レポート生成

### 4-2. 機械学習モデルの精度実績値

| 手法 | Big5予測精度（Pearson r） |
|---|---|
| 従来ML（SVM・ランダムフォレスト） | r = 0.26〜0.39 |
| 非減衰補正後の理論値 | r = 0.39〜0.60 |
| CNN-LSTM（音声のみ） | UAR 74〜80% |

最も予測しやすい因子：外向性（E）・神経症傾向（N）
最も予測しにくい因子：協調性（A）・開放性（O）単体

## 第7章：実装上の重要注意事項

| 注意事項 | 対策 |
|---|---|
| 多変量アプローチが必須 | GeMAPS 62次元 + カスタム特徴量の複合モデルを使用 |
| 性別補正が必須 | 性別入力後にF0・フォルマントを正規化処理 |
| 課題依存性が高い | 自由発話（1〜3分）に統一 |
| 知覚 vs 実際の区別 | 「他者に与える印象」「傾向」として表現する設計 |
| 個人差・文化差 | 「可能性の診断」「インサイト」として誠実に表現 |
| 法的・倫理的配慮 | 誇大表現を避ける。音声データは処理後に自動削除 |
