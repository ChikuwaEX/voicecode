## PDFレポート構成（スマホ縦型・SNSシェア対応）

### 基本仕様
- フォーマット：A4縦型（スマホ表示最適化）
- ページ数：8〜10ページ
- デザイン：ダーク系（深紺×ゴールド）・星空・神秘的
- フォント：日本語メイン

### セクション構成

#### 表紙
```
╔══════════════════════════════╗
║     VOICECODE                ║
║     あなたの声紋診断         ║
║                              ║
║  声紋コード: #A7-432Hz       ║  ← ユニークID（科学性×神秘感）
║  診断日: 2026.06.13          ║
║  ○○様 専用レポート          ║
╚══════════════════════════════╝
```

#### P1: あなたの声紋コード
- 声紋ID（#英数字-周波数）の表示
- 「この宇宙でたった一つの、あなたの声の設計図です」
- 声の主要特徴量の視覚化（レーダーチャート）
- 5元素バランスグラフ

#### P2: あなたの元素タイプ
- タイプ名（大きく・インパクト重視）
- タイプのキャッチコピー（詩的1行）
- 「地球上の○%の方がこのタイプです」（希少性演出）
- タイプのビジュアルシンボル

#### P3: あなたの声が語る真実
- 声の特徴の詩的解説（科学データを自然言語化）
- 例：「あなたの声は432Hzを中心に輝いています。これは…」
- 「多くの方が最初に驚く、あなたの声の秘密」

#### P4: まだ気づいていない才能（★最重要）
- 「あなたが当たり前だと思っている○○は、実は稀有な才能です」
- 才能の具体的な説明（3つ）
- 才能が発揮される場面の例
- 「なぜあなたはそれに気づかなかったのか」の解説

#### P5: あなたの使命のパターン
- 魂の役割の詩的な解説
- 「過去の出来事が持つ意味の再解釈」
- 例：「なぜあなたがあの体験をしたのか、今わかります」

#### P6: あなたの魂が辛いこと（シャドウ）
- 弱点ではなく「使いきれていない力」として表現
- 深い共感を生む表現（「そうなんです…」と思わせる）
- シャドウを超えるためのヒント

#### P7: 共鳴する元素・チャクラ
- あなたのタイプと共鳴する元素色・チャクラ
- 相性の良いタイプ・難しいタイプ
- ※グッズ展開（ストーン・アクセサリー等）への伏線

#### P8: 2026年のあなたへの宇宙のメッセージ
- 「今この時期に声を診断したことの意味」
- 年間のエネルギーと声紋の関係
- タイムリーな気づき（「今が転換点」という演出）

#### P9: あなたの才能を開く3つのカギ
- 具体的な行動指針（興味・職業・人間関係へのアドバイス）
- 「このタイプの方が開花するパターン」
- 次のステップへの誘導（上位プランへの自然なCTA）

#### 裏表紙
- VOICECODEのロゴ・ブランドメッセージ
- 公式LINEへの誘導QRコード
- SNSシェア促進メッセージ

## 疎結合設計：世界観の外部化

### 設計思想
- 世界観・表現はすべてYAMLファイルに格納
- コード（diagnosis_engine.py）は世界観ファイルを読み込むだけ
- YAMLを差し替えるだけで表現を全変更可能

### フォルダ構成
```
src/diagnosis/worldview/
├── worldview_loader.py    ← YAMLを読み込むクラス（変更しない）
└── themes/
    ├── elements_v1.yaml   ← 5元素世界観（今から作る）
    ├── elements_v2.yaml   ← 将来の改訂版
    └── corporate_v1.yaml  ← 法人向け版（将来）
```

### 変更の容易さ
```
世界観を変えたい時: elements_v1.yaml を編集するだけ
新しい世界観を追加: 新しいYAMLを追加するだけ
コードの変更: 一切不要
```
